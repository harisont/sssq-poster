\block{UD Treebanks for SLA Research}{
    Advantages of UD treebanks in corpus-based Second Language Acquisition research:
    \vskip 0.5em
    \begin{itemize}
        \item uniform morphosyntactic annotation layer allowing for qualitative and quantitative \textbf{cross-lingual comparisons}:
        \begin{itemize}
            \item between standard and learner language
            \item between a learner's L1 and L2
            \item between different L2s
        \end{itemize}
            \item possibility to carry out \textbf{grammatical error retrieval and analysis} and \textbf{automatic feedback generation} without explicit error tagging e.g. if learner sentences are paired with corrections (\textbf{\textit{parallel L1-L2 treebank}})
        \item \textbf{semi-automatic annotation} with the help of the existing UD parsers
    \end{itemize}
}
